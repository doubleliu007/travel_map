# 我的旅行地图

一个基于 React + ECharts 的交互式中国旅行足迹地图，展示全国市级行政区划，点击城市即可标记去过的地方。

## 功能

- **全国市级地图**：在一张地图上展示中国所有市级行政区划边界
- **省级边界高亮**：省级边界线加粗显示，便于区分省份
- **点击标记**：点击任意城市标记为"已去过"，再次点击取消
- **数据持久化**：旅行记录自动保存在浏览器 localStorage 中
- **统计面板**：左侧边栏实时展示已去过的省份和城市数量
- **足迹详情**：按省份分组展示所有去过的城市
- **地图交互**：支持滚轮缩放、拖拽移动，缩放后点击不会重置视图
- **直辖市/特别行政区整体显示**：北京、上海、天津、重庆、台湾、香港、澳门作为整体区域展示

## 技术栈

- React 19 + TypeScript
- Vite
- ECharts（地图渲染）
- DataV GeoJSON API（地理数据）

## 开始使用

```bash
npm install
npm run dev
```

打开 http://localhost:5173 即可使用。

## 构建

```bash
npm run build
```

构建产物在 `dist/` 目录下。
